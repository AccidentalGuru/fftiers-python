About



This website is meant to serve as a weekly aid for Fantasy Football. The website includes an interactive sheet for viewing rest-of-season rankings compared to owned players, as well as rest-of-season and weekly positional ranking-graphs.
Players are assigned tiers, for which players within a like-tier can be considered equals. This aid can be used to decide which players to pick-up via free agency and which players to start on a weekly basis. This concept was expanded from Boris Chen's work, which can be found here (http://www.borischen.co/).
This program utilizes unsupervised machine learning by flat clustering with KMeans -- a simple way to uncover like tiers within the player data mined from FantasyPros (http://www.fantasypros.com/).
